# 🏠 House Price Prediction ML Project

An end-to-end Machine Learning project to predict house prices using multiple regression algorithms.

## 🔍 Project Overview
This project compares 4 ML models:
- Linear Regression
- Decision Tree Regressor
- Random Forest Regressor
- XGBoost Regressor

## 📁 Project Structure
| File | Description |
|------|-------------|
| `House_Price_Prediction.ipynb` | ML notebook with EDA and model training |
| `app.py` | Streamlit web application |
| `house_price_model.pkl` | Trained ML model |
| `house_price_prediction_dataset.csv` | Dataset used |
| `requirements.txt` | Required libraries |

## 🛠️ Libraries Used
- Python
- Pandas, NumPy
- Scikit-learn
- XGBoost
- Streamlit
- Matplotlib, Seaborn

## 🚀 How to Run Locally
```bash
git clone https://github.com/Mythili-Pandiyarajan/house-price-prediction.git
cd house-price-prediction
pip install -r requirements.txt
streamlit run app.py
```

## 📊 Model Performance
| Model | R² Score |
|-------|----------|
